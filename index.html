How to Use the Smart Cursor Framework

To integrate the Smart Cursor framework into your project, follow the steps below:

Step 1: Add the Script

Add the following lines to the `head` section of your HTML file:

<script src="https://rubaid07.github.io/smart-cursor/smart-cursor.js" defer></script>

Step 2: Initialize the Cursor

Add the following script to the end of the `body` tag in your HTML file:

<script>
document.addEventListener('DOMContentLoaded', function() {
    initSmartCursor({
        cursorColor: 'rgba(0, 255, 0, 0.6)', // Green cursor
        cursorSize: 30,
        cursorBlur: 2,
        hoverScale: 1.8,
        clickScale: 0.8
    });
});
</script>

Step 3: Enable Cursor Snap on Elements

Add the ` data-smart-cursor-snap="true" ` attribute to the elements you want the cursor to snap to. For example:

<button data-smart-cursor-snap="true">Click me</button>

Users can customize the following features if they want:

  • cursorColor: cursor color
  • cursorSize: cursor size (in pixels)
  • cursorBlur: cursor blur effect (in pixels)
  • hoverScale: scale of the cursor when hovering
  • clickScale: Cursor scale when clicked

These customization options will help users create smart cursors to suit their website design.
